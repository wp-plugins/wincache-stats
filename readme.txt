=== Wincache Stats Widget ===
Contributors: ksingla
Tags: wincache, widget
Stable tag: trunk
Requires at least: 2.8
Tested up to: 2.9

Add a widget to display wincache statistics.

== Installation ==
1. Upload `wincache-stats.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Browse to Appearance->Widgets, add the widget, and configure the options
